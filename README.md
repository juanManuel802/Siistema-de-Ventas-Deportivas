# Siistema de Ventas Deportivas

Laboratorio de procesos Estocásticos, donde se usa teoría de Colas para algoritmo de recomendación de productos según usuario.

This is a code bundle for Minimal Sports E-commerce App. The original design is available at [Figma](https://www.figma.com/design/PUPq2J4su0krgqz6YheiGa/Minimal-Sports-E-commerce-App).

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.
